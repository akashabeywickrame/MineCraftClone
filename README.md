# MineCraftClone
 A voxel-based sandbox game inspired by Minecraft, developed using modern OpenGL. This project focuses on chunk-based world rendering, procedural terrain generation, block placement and destruction, and real-time 3D graphics optimization.

## ⛰️ Perlin Noise terrain. 🌞 Sine wave-driven day/night cycle. 🎯 Laser-pointer style ray casting for block interaction. 📷 Smooth first & third-person cameras.
